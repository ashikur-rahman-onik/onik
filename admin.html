📖 Firebase Setup Guide
Firebase config বসানো ও সব কিছু চালু করার step-by-step নির্দেশনা।
1
Firebase Project তৈরি করো
যাও console.firebase.google.com → "Add project" → নাম দাও: onik-portfolio → Create project।
2
Config Copy করো
⚙️ Project Settings → "Your apps" → </> (Web) → App register করো → firebaseConfig object টা copy করো। এটা index.html ও admin.html দুটোতেই বসাও যেখানে YOUR_API_KEY লেখা আছে।
3
Firestore চালু করো + Rules বসাও
Build → Firestore Database → Create database → Production mode → Region: asia-south1। তারপর Rules tab-এ:
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read: if true;
      allow write: if request.auth != null;
    }
  }
}
4
Storage চালু করো (ছবির জন্য)
Build → Storage → Get started → Production mode → asia-south1। Rules:
rules_version = '2';
service firebase.storage {
  match /b/{bucket}/o {
    match /{allPaths=**} {
      allow read: if true;
      allow write: if request.auth != null;
    }
  }
}
5
Authentication চালু করো
Build → Authentication → Get started → Sign-in method → Email/Password → Enable → Save। ব্যস! এটুকু করলেই admin.html খুললে তুমি নিজেই email+password দিয়ে account বানাতে পারবে।
6
Upload করো ও শুরু করো!
index.html ও admin.html — দুটো file hosting-এ upload করো। তারপর yoursite.com/admin.html খুলো → প্রথমবার নিজের email+password দিয়ে account বানাও → Login করো → সব edit শুরু করো! 🎉
🔒 Security: Firestore rules এমন যে সবাই portfolio দেখতে পারবে, কিন্তু শুধু তুমি (logged in) data পরিবর্তন করতে পারবে — hacker-রা কিছু করতে পারবে না।
📬 Messages: Contact form-এর messages সরাসরি Firebase-এ save হবে ও Messages tab-এ দেখাবে।
